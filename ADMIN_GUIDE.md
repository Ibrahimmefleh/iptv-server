# دليل لوحة التحكم الإدارية

## نظرة عامة

لوحة التحكم الإدارية توفر واجهة رسومية سهلة الاستخدام لإدارة سيرفر البث الخاص بك. يمكنك من خلالها:

- ✅ إيقاف وتشغيل القنوات
- ✅ تغيير روابط المصادر (URLs) للقنوات
- ✅ جدولة تغييرات مؤقتة للروابط
- ✅ إدارة الشريط الإخباري (Ticker)
- ✅ عرض إحصائيات البث المباشرة

---

## الوصول إلى لوحة التحكم

### الرابط
```
https://your-server.com/admin.html
```

### كلمة المرور الافتراضية
```
admin123
```

**تغيير كلمة المرور:**
عدّل متغير البيئة `ADMIN_PASSWORD` في ملف `.env`:
```
ADMIN_PASSWORD=your_secure_password_here
```

---

## أقسام لوحة التحكم

### 1. إدارة القنوات (Channels Management)

#### عرض حالة القنوات
- **الاسم:** اسم القناة
- **الحالة:** ON (نشطة) أو OFF (معطلة)
- **المشاهدون:** عدد المشاهدين الحاليين

#### تبديل القناة
اضغط على زر "إيقاف" أو "تشغيل" لتفعيل/تعطيل البث الفوري للقناة.

#### تحرير القناة
1. اضغط على زر "تحرير"
2. **تغيير الرابط الدائم:**
   - أدخل رابط المصدر الجديد في حقل "رابط المصدر الحالي"
   - اضغط "حفظ التغييرات"
   - سيتم تطبيق الرابط الجديد فوراً

3. **جدولة تغيير مؤقت:**
   - أدخل رابط بديل في حقل "رابط مؤقت"
   - حدد وقت البدء ووقت الانتهاء
   - سيتم تطبيق الرابط البديل في الوقت المحدد
   - سيعود تلقائياً للرابط الأصلي بعد انتهاء الوقت

**مثال عملي:**
- الرابط الأصلي: `http://stream.example.com/channel1`
- رابط مؤقت: `http://stream.example.com/match-live`
- وقت البدء: 2026-07-05 20:00
- وقت الانتهاء: 2026-07-05 23:00

---

### 2. الشريط الإخباري (Ticker)

#### إضافة نص إخباري
1. اذهب إلى تبويب "الشريط الإخباري"
2. أدخل النص الذي تريد عرضه في حقل "النص"
3. حدد عدد مرات التكرار
4. فعّل الخيار "تفعيل الشريط"
5. اضغط "حفظ الشريط"

#### كيفية الظهور
- يظهر الشريط أسفل مشغل الفيديو مباشرة في `watch.html`
- النص يتحرك من اليمين إلى اليسار
- يتكرر حسب عدد المرات المحددة

**أمثلة:**
```
📢 مباراة الدوري الممتاز - الليلة الساعة 8 مساءً
🔴 تحديث: تم تحسين جودة البث
⚠️ صيانة مجدولة الجمعة من 2 إلى 3 صباحاً
```

---

### 3. الإحصائيات (Statistics)

عرض مباشر لـ:
- **إجمالي المشاهدين:** عدد المشاهدين على جميع القنوات
- **القنوات النشطة:** عدد القنوات المفعلة
- **مشاهدو كل قناة:** توزيع المشاهدين على القنوات

تُحدّث الإحصائيات تلقائياً كل 5 ثوانٍ.

---

## حالات الاستخدام الشائعة

### حالة 1: تشغيل مباراة مباشرة بروابط مختلفة

**السيناريو:** تريد بث مباراة كرة قدم على قناة معينة لمدة ساعتين.

**الخطوات:**
1. اذهب إلى "إدارة القنوات"
2. اضغط "تحرير" للقناة المطلوبة
3. أدخل رابط المباراة في حقل "رابط مؤقت"
4. حدد وقت البدء (مثلاً: 20:00)
5. حدد وقت الانتهاء (مثلاً: 22:00)
6. اضغط "حفظ التغييرات"
7. السيرفر سيطبق الرابط تلقائياً في الوقت المحدد

### حالة 2: إيقاف قناة للصيانة

**الخطوات:**
1. اضغط على زر "إيقاف" بجانب القناة
2. سيتم قطع البث فوراً
3. سيرى المشاهدون رسالة "Channel is offline"
4. عند الانتهاء، اضغط "تشغيل" لاستئناف البث

### حالة 3: إضافة إعلان في الشريط الإخباري

**الخطوات:**
1. اذهب إلى "الشريط الإخباري"
2. أدخل النص: `🎬 حفل توزيع الجوائز - الجمعة 8 مساءً`
3. حدد عدد مرات التكرار: 3
4. فعّل الشريط
5. اضغط "حفظ الشريط"
6. سيظهر الشريط تحت كل مشغل فيديو

## الأمان

### نصائح الأمان

1. **غيّر كلمة المرور الافتراضية فوراً**
   ```
   ADMIN_PASSWORD=your_very_secure_password_123
   ```

2. **استخدم HTTPS**
   - تأكد من أن السيرفر يعمل على HTTPS
   - لا تستخدم HTTP للوصول إلى لوحة التحكم

3. **قيود الوصول**
   - استخدم جدار حماية (Firewall)
   - قيّد الوصول إلى `/admin.html` من عناوين IP محددة

4. **المراقبة**
   - راقب السجلات بانتظام
   - تحقق من الوصول غير المصرح

## استكشاف الأخطاء

### المشكلة: لا يمكن الوصول إلى لوحة التحكم

**الحل:**
- تحقق من أن السيرفر يعمل: `http://your-server.com/health`
- تحقق من كلمة المرور
- امسح ذاكرة التخزين المؤقت (Cache)

### المشكلة: التغييرات لا تُطبق فوراً

**الحل:**
- انتظر 5-10 ثوانٍ للتحديث
- أعد تحميل صفحة البث
- تحقق من اتصال الإنترنت

### المشكلة: الشريط الإخباري لا يظهر

**الحل:**
- تأكد من تفعيل الشريط (الخيار مفعّل)
- تأكد من إدخال نص في الحقل
- أعد تحميل صفحة البث

## API المتقدمة

إذا كنت مطوراً، يمكنك التحكم في السيرفر برمجياً:

### الحصول على إحصائيات القنوات
```bash
curl http://your-server.com/api/stream/stats
```

### إيقاف/تشغيل قناة
```bash
curl -X POST http://your-server.com/api/admin/channel/channel1/toggle \
  -H "x-admin-token: admin123"
```

### تحديث رابط القناة
```bash
curl -X PUT http://your-server.com/api/admin/channel/channel1/source \
  -H "x-admin-token: admin123" \
  -H "Content-Type: application/json" \
  -d '{"source":"http://new-url.com/stream"}'
```

### جدولة تغيير مؤقت
```bash
curl -X POST http://your-server.com/api/admin/channel/channel1/schedule \
  -H "x-admin-token: admin123" \
  -H "Content-Type: application/json" \
  -d '{
    "tempSource":"http://temp-url.com/stream",
    "startTime":"2026-07-05T20:00:00Z",
    "endTime":"2026-07-05T23:00:00Z"
  }'
```
